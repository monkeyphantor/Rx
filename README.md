# Rx
C++ Game Engine powered by flecs, Vulkan and Jolt Physics


### Platforms
1.] Windows \
2.] Mac \
3.] Linux \
Just clone vcpkg and install glm, glfw, vulkan and Jolt Physics and build with CMake


### Examples
1.] PBR Cubes with one draw call per Cube \
2.] Instanced Cube World \
3.] Physics Cubes
